# ML Based Login Attack Detection and Database Forensics System

## Project Overview
This project detects suspicious login activities using Machine Learning and performs database forensics for security investigation. The system monitors login behaviour, identifies anomalies and records evidence for further analysis.

## Objectives
- Detect brute force login attacks
- Identify unusual login behaviour
- Perform database log analysis
- Support cyber security incident investigation

## Technologies Used
- Python
- Machine Learning
- MySQL / Database Logs
- Cyber Security Concepts

## Methodology
1. Collected login activity dataset
2. Preprocessed and cleaned the data
3. Trained Machine Learning model to detect anomalies
4. Detected suspicious login attempts
5. Performed database forensic analysis on logs

## Features
- Login anomaly detection
- Attack alert generation
- Database log investigation
- Evidence collection for forensics

## Skills Demonstrated
- Machine Learning for Security
- Intrusion Detection
- Database Forensics
- Python Programming

## Conclusion
This project demonstrates the use of Machine Learning in cyber security to detect login attacks and support digital forensic investigations.
